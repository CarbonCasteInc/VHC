# Documentation Archive

Files under `docs/archive/` are historical execution snapshots. They preserve
audit context but do not describe current implementation or operational state.

Use `docs/README.md` and `docs/CANON_MAP.md` to find current authoritative
documents. An archived file must name its current replacement near the top and
must not be used as live execution authority.

## Archive Bundles

- `public-beta-pre-recovery-2026-07-10/` preserves the exact status, launch
  closeout, handoff, and orchestration checklist that existed immediately before
  the first supervised A6 load attempt, plus superseded May-July release and
  Phase 5 evidence reports. Its transition prose is historical.
